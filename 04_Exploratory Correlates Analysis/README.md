# 04_Exploraty Correlates Analysis
This folder contains R Markdown scripts for exploratory analyses, including trajectory differences in structural imaging, baseline cognitive, and longitudinal functioning data.

## Script Summaries
- brain.Rmd: Creates brain maps displaying structural brain differences across depressive symptom trajectory classes via pairwise comparisons.
- sans_brain.Rmd: Creates brain maps displaying structural brain differences across negative symptom trajectory classes via pairwise comparisons.
- functioning_cog.Rmd: Conducts non-parametric analyses to compare baseline cognition and longitudinal functioning (SOFAS) across symptom trajectories.
